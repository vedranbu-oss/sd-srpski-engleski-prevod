# Srpski → Engleski prevod prompta (Stable Diffusion WebUI)

Ekstenzija prevodi korisnički prompt sa srpskog na engleski pre generisanja slike.

## Instalacija
1. Otvori Stable Diffusion WebUI → Extensions → Install from URL
2. URL: https://github.com/<tvoj-nalog>/<ime-repozitorijuma>
3. Klikni Install → Apply and restart UI
4. Označi opciju "Prevedi prompt sa srpskog na engleski".

## Zahtevi
- `pip install googletrans==4.0.0-rc1`

## Napomena
Za kvalitetniji prevod možeš koristiti DeepL API (plaćena opcija).
